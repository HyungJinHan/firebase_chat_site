## 1. Usage

  - Dependencies 설치법
    - `npm` 버전 : 9.3.1 (Latest Release)
    - `Node.js` 버전 : v18.13.0 (LTS)

    <br>

    ```
    # 순서에 맞춰서 진행
    
    # 1. Clone한 프로젝트의 폴더로 이동
    $ cd firebase_chat_site

    # 2. npm을 통한 패키지 설치
    $ npm install # or npm i
    ```
    
  - 사용

    ```
    # 3. 서버 실행 (http://localhost:8888)
    $ npm start # or npm run dev
    ```
